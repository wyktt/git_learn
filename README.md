# 俄罗斯方块 (Tetris)

一个经典的俄罗斯方块游戏，使用 HTML5 Canvas + JavaScript 开发，可以直接在浏览器中运行。

## 🎮 在线游玩

访问 GitHub Pages 即可直接游玩！

## 🚀 功能特性

- 经典俄罗斯方块玩法
- 下一个方块预览
- 分数、等级、消行统计
- 渐进式难度（速度随等级提升）
- 精美的 UI 设计
- 响应式布局，支持移动端

## 🎯 操作说明

| 按键 | 功能 |
|------|------|
| ← → | 左右移动 |
| ↑ | 旋转方块 |
| ↓ | 加速下落 |
| 空格 | 直接落下 |
| P | 暂停/继续 |

## 📁 项目结构

```
├── index.html    # 主页面
├── style.css     # 样式文件
├── tetris.js     # 游戏逻辑
└── README.md     # 说明文档
```

## 🛠️ 本地运行

直接在浏览器中打开 `index.html` 文件即可开始游戏。
